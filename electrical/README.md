# Electrical Engineering Subsystem (Flight Controller PCBA)

## Motivation
To design a highly robust, EMI-shielded 4-layer Power Distribution Network (PDN) and microcontroller logic board. This board must survive the extreme $di/dt$ inductive spikes generated by 40A ESC switching while providing mathematically clean SPI data to the ESP32 for 1000Hz PID loop execution.

## Architecture (Hierarchical Sheets)
We will use a Top-Down Hierarchical design in KiCad to keep the schematic industry-grade.
1. **`top_level.sch`**: The master block diagram connecting the sub-sheets.
2. **`power_distribution.sch`**: 5V to 3.3V LDO Regulator, LC filtering, and bulk/decoupling capacitors.
3. **`mcu_esp32.sch`**: The ESP32 logic, strapping pins, and UART/Programming headers.
4. **`sensors_imu.sch`**: The BMI270/MPU6000 IMU, SPI bus wiring, and strict decoupling.
5. **`io_connectors.sch`**: The ESC DSHOT pads, battery voltage sensing voltage-divider, and ESP-NOW RX pins.

## Critical Formulations
- **LC Filter Resonance (For IMU Power Cleanliness):** 
  $f_{res} = \frac{1}{2\pi \sqrt{L C}}$
  We must ensure $f_{res}$ is well below the 24kHz PWM frequency of the ESC to block high-frequency noise.
- **Trace Width Ampacity (IPC-2221):**
  $I = k \cdot \Delta T^b \cdot A^c$ 
  5V traces must handle 1A. Signal traces handle mere mA.
- **EMI Loop Area Minimization:**
  $V_{noise} = - \frac{d}{dt} \int \mathbf{B} \cdot d\mathbf{A}$
  Solid GND plane placed exactly 0.1mm below Layer 1 signal traces ensures $d\mathbf{A} \to 0$.
